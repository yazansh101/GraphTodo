# GraphTodo
Graph Todo is a simple todo app built using GraphQL and Provider package.
# Features
- Add tasks with descriptions and priorety.
- Mark tasks as completed or incomplete.
- Delete tasks.
# Technologies Used
GraphQL
Provider Package
